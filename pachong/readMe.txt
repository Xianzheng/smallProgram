1装python
我用的Python version 3.7.6
以下的一切代码都可以在cmd中运行
window + r 打开cmd， cd 到当前目录

2在该文件夹下创建虚拟环境
python -m venv venv

3进入虚拟环境
./venv/Scripts/activate

4安装支持库
pip install -r requirements.txt

5测试，
进入ios,cd ios，运行python combine.py 运行结果就会出现在result中

进入android,cd android，运行python combine.py 运行结果就会出现在result中
